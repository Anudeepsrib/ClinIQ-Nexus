THE NON-NEGOTIABLES

Governance is
not a policy document.

It is code that executes on every single request, enforced at multiple independent layers.

  • No clinical output reaches a user without passing MCP + (if required) human review
  • Every PHI touch, routing decision, and review action is written to an immutable audit log
  • Memory and Deep Agents are strictly governed — no raw clinical context retained without minimization
Core Invariant
"There is no path from user input to model inference that bypasses the MCP Governance layer."
This is not aspirational. It is enforced in middleware, service boundaries, and LangGraph checkpoints.
